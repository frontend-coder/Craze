HTML Craze

HTML Craze - является предсатвителем шаблонов эконом-класса.

Сверстан только decktop вариант шаблона, если вы желаете использовать этот шаблон в своей деятельности, вы его можете адаптировать
своими силами или заказать доработку у верстальщика HTML Craze.

После внесения изменений в код HTML шаблона Craze, за правильность работы данного шаблона ответсвенности не несу.
